A modern and interactive cybersecurity portfolio built to showcase my skills, projects, and experience in security analysis, threat detection, and web security.

## About

This portfolio highlights my journey as a Cybersecurity Analyst and Security Enthusiast. It includes my technical skills, cybersecurity projects, and professional contact information in a visually engaging 3D interface.

## Features

* Responsive modern UI
* Interactive 3D hero section
* Smooth animations and transitions
* Cybersecurity-focused branding
* Skills and project showcase
* Social media integration
* Mobile-friendly design

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Framer Motion
* JavaScript / HTML / CSS

## Projects Included

* Web Security Assessment Platform
* Security Research Projects
* Cybersecurity Labs and Experiments
* Additional Security Tools and Case Studies
